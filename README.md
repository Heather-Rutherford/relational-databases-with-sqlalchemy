## Advanced Python Module Project: Defeat the Evil Wizard

## Overview

Practice creating and managing a relational database using Python and SQLAlchemy. You will define tables, set up relationships, and perform basic CRUD operations.

## Requirements

- Python 3.7 or higher

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Heather-Rutherford/relational-databases-with-sqlalchemy.git
   ```
2. Navigate to the project directory:
   ```bash
   cd relational-databases-with-sqlalchemy
   ```

## Running the Application

Run the application using Python:

```bash
python relational-databases.py
```
